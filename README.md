# Folk Artist Website

A simple, responsive website for a local indie/folk acoustic guitar player.

## Features

- Clean, minimalist design with quirky line drawings in the background
- Responsive layout that works on all devices
- Three main sections: About, Videos, and Contact
- Smooth scrolling navigation
- Background illustrations using Lucide icons
- Subtle parallax effect on background elements

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [Lucide Icons](https://lucide.dev/) for the background illustrations

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser

## Customization

### Changing Content

- Edit the text in `index.html` to update the artist's information
- Replace the YouTube video embed links with actual performance videos
- Update contact information in the contact section

### Styling

- Colors, fonts, and layout can be customized in `styles.css`
- Background icons can be changed by editing the icons in the `.background-icons` div in `index.html`

### Adding More Sections

To add more sections:

1. Add a new section in `index.html` following the existing pattern
2. Add a corresponding navigation link in the header
3. Style the new section in `styles.css`
